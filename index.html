Primeiro ato -

17/12/2023

A primeira postagem deste blog não poderia versar sobre algo senão o próprio conteúdo do blog. No momento desta redação, foram utilizadas exclusivamente as linguagens HTML e CSS. Permita-me detalhar cada aspecto. O HTML, uma linguagem de marcação, é empregado na construção de páginas da Web. Documentos HTML são interpretados pelos navegadores, sendo, assim, a espinha dorsal na criação dos sites com os quais está familiarizado. Não é minha intenção, nesta ocasião, aprofundar-me excessivamente nesse tema, mas gostaria de disponibilizar um link para que possa explorar mais sobre HTML, (ler mais...). Em continuidade, temos o CSS, um mecanismo destinado a adicionar estilos às páginas web. Estes estilos podem ser aplicados diretamente nas tags HTML ou contidos dentro das tags <style>. Outra abordagem é a inclusão de estilos através de um link para um arquivo CSS que engloba os estilos, método que adotei na concepção deste blog, por sua natureza mais organizada. Em termos simplificados, o CSS estiliza ou aprimora o código HTML em desenvolvimento. Sua criação remonta a 17 de dezembro de 1996 e, desde então, tornou-se praticamente indispensável na criação de páginas web, aliado ao HTML.
